  • 5.3.5.1. Подготовка каналов для анемостатов

    Необходимая документация для выполнения работ: Разделы ИР.Вентиляция, ЭОМ, Раздел № 14.4 Схема устройства вентканала с анемостатами.

    Производимые операции:

      Наличие в модуле электрощита, РК №2 или РК №3 указано в разделе ЭОМ.
    1. Смонтировать электрощит универсальной конструкции с вырезами и колодцами для кабелей. Монтаж щита происходит в 3 этапа.
      1. Установить щит в проектное положение, совместить с вырезами для кабелей, закрепить саморезами.
      2. Вставить колодцы в вырезы для кабелей и закрепить саморезами через отверстия в "ушках".
      3. Оставшееся отверстие закрыть заглушкой (поставляется в комплекте со щитом) и закрепить заглушку вытяжными клепками.
      Монтаж "РК №2" для главной распайки дома производится аналогично электрощиту.
    2. Смонтировать металлическую "РК №3" для расключения кабелей. Монтаж данной РК выполняется в следующей последовательности.
      1. В ТСП сделать аккуратный вырез под РК.
      2. По периметру юбки РК приклеить d-профиль. Расположить d-профиль на внутренней стороне юбки вплотную к корпусу РК.
      3. Вставить РК в вырез. Отверстие для кабелей должно быть направлено вверх. Закрепить РК саморезами ПШО 4,2х16 (допускается ПШО 4,2х19). Юбка РК не должна доходить до ТСП на 2мм. Корпус РК внутри модуля не должен выступать за пределы ТСП. Иначе при установке крышки РК будет щель между ТСП и крышкой.
        4. Руководство по монтажу "РК металл №3"
    3. Установить гильзы для ввода кабельных линий. Гильзы обязательно зафиксировать саморезом для предотвращения их смещения в процессе монтажных работ и при эксплуатации дома. У резервных гильз из труб ПП50, которые будут закрываться заглушками, необходимо извлечь резиновое уплотнительное кольцо, т.к. оно со временем прикипает к заглушке, которую после этого довольно проблематично извлечь из трубы.

    Контрольные параметры:

    • Расположение электрощита, РК, гильз соответствует проекту.

    Необходимый инструмент и оборудование:

    • Электролобзик с пилкой по дереву.
    • Инструмент электрика.
    • Аккумуляторный шуруповерт с зашимым патроном.
    • Бита PZ2.
    • Сверло перьевое 12мм.
    • Заклепочник для вытяжных клепок.
    • Рулетка 5м.
    • Плотницкий уголок металлический 250мм.
    • Карандаш простой.

    Расходные материалы:

    • Саморез ПШО 4,2х16 (ПШО 4,2х19).
    • Клепка вытяжная 4х15мм.

  • 5.3.4.2. Проклакдка кабельных линий

    Необходимая документация для выполнения работ: Раздел КР.М, ЭОМ.

    Производимые операции:

      Перед началом работ ознакомиться с пояснительной запиской раздела ЭОМ.

    1. Нарезать кабель на отрезки, согласно раздела ЭОМ. Нанести маркером наимаенование кабельной линии на концах отрезков.
    2. Проложить кабель в модуле, согласно листа ЭОМ. Кабель прокладывается по кровле через специальные отверстия в стропилах. К каждой электроточке, расположенной на стенах, кабель опускать вертикально. Максимальное отклонение от вертикальной оси электроточки не более 150мм.
      Запас кабеля на расключение электроточки оставлять 300мм. Если электроточка запитывается "шлейфом"кабель приходит к этой электроточке и после расключения уходит к следующей, то запас кабеля делать по 300мм на каждую электроточку.
      Если подписанные концы проводов отрезаются, то необходимо продублировать обозначение линии строительным маркером на оставшемся конце кабеля.
      Прокладываемые параллельно кабели между собой в пучке крепить пластиковыми стяжками. Линии, расположенные на стене, фиксировать с помощью киперной ленты, закрепленной к стене скобами А-06.
    3. Питающий кабель выводится в гофре

    Контрольные параметры:

    • Концы кабельных линий подписаны.
    • Кабельные линии по стенам проложены вертикально.
    • Отклонение кабельной линии на стене от вертикалной оси электроточки не более 150мм.
    • Кабельные линии по стенам закреплены киперной лентой.

      • По факту контрольных замеров ставится отметка в паспорте ОТК.

      По итогам законченного этапа сделать фотоотчет выполненных работ.
      На фотоотчете необходимо показать: последовательные фотографии участков кабельной трассы, чтобы можно было отследить схему прокладки до каждого потребителя. На каждом фото должна быть табличка с ID объекта, не перекрывающая информацию фотоотчета.

    Необходимый инструмент и оборудование:

    • Инструмент электрика.
    • Пневмостеплер.
    • Рулетка 5м.
    • Плотницкий уголок металлический 250мм.
    • Карандаш простой.

    Расходные материалы:

    • Скоба А-06.
    • Киперная лента.
